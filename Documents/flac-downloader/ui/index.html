What leaves your machine:
 • Odesli (api.odesli.co) — source URL, only when DRM fallback triggers
 • Proxy services — numeric Qobuz track ID only, when DRM fallback triggers
 • MusicBrainz — title + artist for ISRC lookup only
 • yt-dlp — requests go directly to the source site

All connections use HTTPS. No account credentials are required or stored.
No analytics or telemetry is collected by this app.
